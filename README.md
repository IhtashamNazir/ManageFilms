# ManageFilms
NestJS task to manage films and search on them
# Prereq to run this code
  Node installed
  Docker installed
  Docker compose installed
  Free Port 8000 and 9200

# To Run
  cd managefilms-micro

  cd gateway
  npm install

  cd ../userAccount
  npm install
  
  cd ..
and run command
  docker compose up

  wait for all the services to up

  Go to http://localhost:8000/api
